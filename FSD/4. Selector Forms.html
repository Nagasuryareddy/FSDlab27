CSS Selector Types & Definitions 1. Simple Selectors Element Selector: Selects all elements of a specific type (e.g., h1 for all

). ID Selector: Selects an element with a specific ID (#id). Class Selector: Selects all elements with a given class (.class). Group Selector: Applies styles to multiple elements at once (h1, p, div). Universal Selector: Selects all elements (*). 2. Combinator Selectors Descendant Selector (A B): Selects all B elements inside A, at any depth. Child Selector (A > B): Selects direct B children of A. Adjacent Sibling Selector (A + B): Selects the immediate next sibling B of A. General Sibling Selector (A ~ B): Selects all B siblings of A, not just the immediate one. 3. Pseudo-Class Selectors Select elements in a specific state (e.g., :hover for when an element is hovered). 4. Pseudo-Element Selectors Style specific parts of an element (e.g., ::first-letter for the first letter of a paragraph). 5. Attribute Selectors Select elements based on attribute values (input[type="text"]). Html code----------------------------- CSS Selectors Demo
